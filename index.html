What does Frontend really mean?

It’s the users experience directly, and it's all about how the web page looks.
That's why you need coding and styling to make the web page stand out.
A web page can’t only be fun and exciting it has to be stylish and functional as well.

The tools you need to know and mainly use are:

HTML

HTML is the coding language that will create, and structure the web content so it can be displayed in the browser.
You can make a very basic web page just with HTML, but that wouldn't be very interesting for the users experience.

CSS

CSS is used to style the HTML elements at your web page. It’s a tool to define layout, colors and fonts.

JAVASCRIPT

JS is used for more interactive elements such as dropdown menu’s, contact forms, modal window etc.
JavaScript is executed in the browser and it makes the web page more dynamic.
